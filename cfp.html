Details

be passionate! be inclusive! here's some other details about speaking:

  • talks are 20min long
  • no Q&A
  • selection process is anonymized and then curated for diversity
  • talks will be recorded and publicized on YouTube
  • all submissions must adhere to our Code of Conduct
  • we are able to cover travel to the conf, the conf ticket, and two nights accomodation in the Bay Area

npmCamp's goal is to give the amazing community built around npm a time to share with each other. to that end, we're looking for talks that:

  • tell a story about how npm helped you build cool software
  • explain a neat-o, oft-unused, feature of npm
  • beginner experiences (e.g. babby's first package!)
  • npm where you wouldn't expect it
  • tools built with and/or on top of npm
  • thoughts and musings on npm's effect on the community
